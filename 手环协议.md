## 一、BLE 连接参数

| 参数 | 值 |
|------|-----|
| Service UUID | `6E40FFF0-B5A3-F393-E0A9-E50E24DCCA9E` |
| 写入特征（发命令） | `6E400002-B5A3-F393-E0A9-E50E24DCCA9E`（Write） |
| 通知特征（收回包） | `6E400003-B5A3-F393-E0A9-E50E24DCCA9E`（Notify） |
| 广播过滤 UUID | `0000FFF0-0000-1000-8000-00805F9B34FB`（扫描时 advertisServiceUUIDs 过滤） |
| 帧结构 | **16 bytes**：前 15 字节为数据，第 16 字节为校验和（前 15 字节之和的低 8 位） |
| 数据格式 | ArrayBuffer，内部转 hex 字符串处理（ab2hex） |
| Write 类型 | `writeType: "write"`（非 write-without-response） |

## 二、下行命令（App → 手环）

### 校验和算法
```js
function checksum(arr15) {
  let sum = 0;
  for (let i = 0; i < 15; i++) sum += arr15[i];
  arr15.push(sum & 0xFF);
  return arr15;
}
```

### CMD 01 — 同步时间
```
01 YY MM DD HH mm SS 00 00 00 00 00 00 00 00 CS
```
YY-SS 均为 BCD 编码：`BCD(v) = 16*(v/10|0) + v%10`，年取后两位。

### CMD 03 — 查询电量
```
03 00 00 00 00 00 00 00 00 00 00 00 00 00 00 03
```

### CMD 0A — 读取/写入设备设置
读取：`0A 01 00×13 0B`  
写入：`0A 02 bb cc dd ee ff gg hh ii jj 00×4 CS`

| 字段 | 含义 |
|------|------|
| bb | 时间制：01=12h，00=24h |
| cc | 单位：00=公制，01=英制 |
| dd-jj | 其他配置（从设备回包缓存，写入时原样透传） |

### CMD 15 — 查询历史心率（按天）
```
15 T3 T2 T1 T0 00×10 CS
```
T0-T3 = 目标日 00:00:00 Unix 时间戳（秒），大端序，支持 N-1 天偏移。

### CMD 16 — 心率定时开关
```
16 xx yy 00×12 CS
```
xx=01 读状态；xx=02 设置（yy=01 开，yy=02 关）。

### CMD 44 — 查询睡眠数据（按天）
```
44 NN 0F 00 96 00×10 CS
```
NN = 天偏移（0x00=今天，0x01=昨天…），0F=15，96=150（固定参数）。

### CMD 69 — 实时测量心率（单次）
```
69 01 00×13 6A   （绑定页）
69 04 00×13 6D   （运动页，每 35 秒轮询）
```

## 三、上行回包解析（手环 → App）

### 0x01 — 时间同步 ACK
`slice(0,2)=="01"` → 触发查电量

### 0x03 — 电量
`parseInt(slice(2,4), 16)` = 电量 %

### 0x0A — 设置回包
`slice(4,6)`=bb … `slice(20,22)`=jj，9 个配置字节全部缓存，用于下次写入。

### 0x15 — 历史心率数据帧（24 帧一组）
```
首帧  slice(2,4)=="01"：时间戳=slice(4,12) 逆字节重组，心率数据=slice(12,30)
扩展帧 slice(2,4)=="17"：心率数据=slice(4,16)
普通帧 cmd>0x01：心率数据=slice(4,30)
```
每个心率值 1 byte：0x00/0xFF=无效（→"000"），有效值补齐 3 位十进制字符串。  
结束：`slice(2,4)=="ff"` → 该天完毕。

### 0x16 — 心率开关 ACK
```
slice(2,4)="01",slice(4,6)="01" → 读到已开启
slice(2,4)="01",slice(4,6)="02" → 读到已关闭
slice(2,4)="02"                  → 写入 ACK，重读确认
```

### 0x44 — 睡眠数据帧（44 帧一组）
```
slice(2,4)=="f0" → 时间帧：slice(6,8)=YY, slice(8,10)=MM, slice(10,12)=DD
slice(2,4)=="2a" → 短睡眠帧：数据=slice(4,16)
其他             → 长睡眠帧：数据=slice(4,38)
```
结束：`slice(0,2)=="c4"` 或 `slice(2,4)=="ff"` → 进入心率同步。

### 0x69 — 实时心率回包
`parseInt(slice(6,8), 16)` = 心率 bpm（byte[3]）

## 四、同步状态机

```
连接成功
  → [10%] CMD 01 同步时间
  → [20%] CMD 03 查电量
  → [40%] CMD 0A 读设置
  → [50%] CMD 16 读心率开关
    ├─ 已开 → [60%] CMD 44 取睡眠（按天循环）
    └─ 未开 → CMD 16 开启 → 重读确认
  → [80%] CMD 15 取历史心率（按天循环，24帧/天）
  → [100%] 上传服务器
```
